# Cognitive Questionnaire PoC

This proof of concept demonstrates the feasibility of integrating a short
MMSE/MoCA-inspired cognitive questionnaire into the VESTA application.

## Purpose
- Validate questionnaire flow and usability
- Demonstrate accessibility-focused UI concepts
- Capture responses for a single assessment session

## Scope
- 8–12 representative questions
- One question per screen
- Single-touch interaction
- No clinical scoring or diagnosis

## Out of Scope
- Full MMSE/MoCA implementation
- Clinical interpretation
- Long-term analytics
